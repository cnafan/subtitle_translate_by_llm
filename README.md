# Subtitle Translate by LLM (Qt Version) 🎬

这是一个基于 **PyQt6** 构建的高性能本地 LLM 批量字幕翻译工具。它专为影视爱好者和字幕组设计，通过连接本地 API（如 LM Studio, vLLM, Ollama），利用大语言模型（如 Qwen 3.5, Llama 3）对字幕进行深度语境感知的翻译。

## ✨ 核心特性

-   **🍎 极简苹果风界面**：采用现代化的 Apple Style Dark Mode 设计，支持磨砂感卡片布局与平滑交互动画。
-   **🎨 电影级 ASS 特效**：
    *   一键生成双语 ASS 字幕。
    *   **中文字幕**：白色，置于上方，清晰易读。
    *   **英文字幕**：经典亮黄色 (`&H00FFFF`)，自动缩小字号 (`\fs38`)，置于下方。
-   **🧠 深度语境翻译**：
    *   **地道化处理**：内置针对美剧、电影优化的资深专家 Prompt，精准处理俚语 (Slang) 与习语 (Idiom)。
    *   **专有名词保护**：智能识别并保留人名、地名等专有名词为英文原文。
-   **🛡️ 工业级稳定性 (Robustness)**：
    *   **自愈式解析**：支持多种标号格式 (`<1>`, `<1]`, `<1)`, `<1:`)，无惧模型幻觉。
    *   **自动去回显**：内置后处理算法，自动识别并剔除 AI 误输出的英文原文。
    *   **逐行补译**：若批量翻译中出现缺失行，系统会自动触发 1-on-1 补偿机制确保零漏行。
-   **🔍 全量调试日志**：
    *   自动维护 `/logs` 目录，按会话生成独立日志文件。
    *   **原子级记录**：将 Prompt、Raw Response 与解析预警归组记录，方便复盘分析。
-   **⚡ 高性能并发**：支持自定义并发请求数与批处理大小 (Batch Size)，完美适配各种显存等级。

## 🛠️ 环境要求

-   **操作系统**: Windows 10/11
-   **Python**: 3.10+
-   **后端驱动**: 建议使用 [LM Studio](https://lmstudio.ai/) 或 [vLLM](https://github.com/vllm-project/vllm) 开启 OpenAI 兼容接口。

## 🚀 快速开始

### 1. 安装依赖

```bash
pip install -r requirements.txt
pip install openai pysrt PyQt6 requests
```

### 2. 启动 API 服务

以 LM Studio 为例：
1. 下载并运行 Qwen 3.5 9B 或 Llama 3 等模型。
2. 开启 **Local Server**，确保端口（默认 1234）可用。

### 3. 运行项目

```bash
python main.py
```

## 📖 使用指南

1.  **准备文件**：将待翻译的英文 SRT 字幕重命名为 `*.en.srt` 后缀。
2.  **设置路径**：在界面选择字幕所在的文件夹。
3.  **配置参数**：
    *   **并发数**：建议 8GB 显存设为 1-2，24GB 显存可设为 4-8。
    *   **输出 ASS**：勾选此项以获得带样式的双语字幕文件。
4.  **开始任务**：点击“开始批量翻译”，程序将自动忽略已翻译文件并输出 `.srt` 和 `.ass` 结果。

## ⚙️ 技术细节

-   **核心驱动**: `PyQt6`, `pysrt`, `requests`
-   **文件策略**: 递归扫描 `*.en.srt` -> 输出 `*.srt` (文本) & `*.ass` (样式)。
-   **日志系统**: 记录在根目录 `/logs` 下，包含完整的 Prompt 与 API 原始响应回执。
